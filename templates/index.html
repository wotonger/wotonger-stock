Stock Web

基于 Django + MySQL 的股票数据可视化系统

项目简介

本项目是一个基于 Django 框架开发的股票数据可视化系统， 集成 MySQL 数据库存储与 ECharts 图表库进行数据展示。 支持K线图、折线图、成交量柱状图等多种图表类型。

技术栈

Python 3.x Django 4.x MySQL ECharts HTML/CSS/JavaScript AJAX

主要功能

• 股票实时数据展示与查询
• ECharts 交互式 K 线图 / 折线图
• 成交量柱状图分析
• MySQL 数据库持久化存储
• Django Admin 后台管理界面
